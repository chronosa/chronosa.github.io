ARRAY

BASH에서는 Array를 사용하지 않는 편이다.
실제로 쓸 일이 별로 없기도 하고, 맨 처음에 Shell을 배울 때 순수한 Bone Shell의 기능만을 가급적 사용하라는
멘토님의 말을 많이 들어서 그런 것 같다.

그래도, 가끔 BASH 코드를 보면 Array가 쓰이는 경우가 있어서 아래정도의 사용 예는 알아두면 좋을 것 같다.
