cloudformation_3 (CloudFormer)
CloudFormerCloudFormer란?Public Document에는 아래와 같이 정의되어 있음…CloudFormer는 계정의 기존 AWS 리소스에서 AWS CloudFormation 템플릿을 생성하는 템플릿 생성 베타 도구입니다. 계정에서 실행 중인 지원되는 AWS 리소스를 선택하면 CloudFormer가 Amazon S3 버킷에 템플릿을 생성합니다.…Docs 다만 아직 Beta라서 PRD에는 적용을 가급적 삼가고, 적용하더라도 소스코드 전수점검 및 사전 테스트가 필요할 것 같다.
